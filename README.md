# Sales-Forecasting
### Background
This project is part of the Tripleten data science practicum. This project focuses on data analysis for sales forcasting.

## Description

This project is tasked with analysis of open source video gaming data on behalf of "Ice" online video game store. 
The project will focus on the analysis of the provided dataset with the goal of identifying patterns across categories to plan a year 2017 campaign.

### Dataset features:

- Name (name of game)
- Platform (platform of game release)
- Year of release (year of game release)
- Genre (game genre)
- Na_Sales (North America sales in USD million)
- Eu_Sales (Sales in Europe USD in million)
- Jp_sales (Sales in Japan in USD million
- Other_sales (Sales in other countries in USD million)
- Critic_score (maximum of 100)
- User_score (maximun of 10)
- Rating (ESRB Rating)

Data for 2016 is specified to be potentially incomplete.

## Objectives

Most sold game globally and by region.
Highest critic and user scores in relation to sales.
Distribution patterns in relation to platforms / sales.
Regional sales profiles: Top 5 platforms / genres.
Statistical comparision between ratings and platforms / genres.
